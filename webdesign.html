Website Design Samples

Senior Capstone - Website Rebrand and Redesign using Figma and Adobe Illustrator

For my final senior capstone project for my Professional Writing and Technical Communications certificate, we were tasked to learn a new software and implement it into a real-world scenario. For my project, I chose to rebrand and redesign the website for an independant, local bookstore in Amherst, Massachusetts named Amherst Books.

I used Adobe Illustrator to create different logo mock-ups and color schemes before implementing these design elements into Figma to create a website prototype. I really enjoyed this assignment and feel that it expanded my knowledge of website design beyond just HTML and CSS.

MadCap Flare Project

As a deliverable for my Advanced Technical Communications course, we were challenged to create a website about Digital Accessibility using MadCap Flare. In collaboration with my classmates, we compiled information on digital accessibility and information on how to make video content more accessible for users.

I then integrated all of this information into my website using MadCap Flare and GitHub for a majority of the creation process. This project helped me learn so much about content development platforms!
